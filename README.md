# Stack Tecnológico

- Frontend: React
- Backend: NestJS
- Base de datos: PostgreSQL
- Autenticación: JWT
- Biometría: WebAuthn

# Arquitectura

API REST

# Repositorio

- Sera un solo repositorio
- Usaremos 3 ramas: dev, test, main, el flujo sera el siguiente:
    - Desarollo normal en develop
    - Hacer pr a test
    - El workflow hace merge a test
    - Se ejecuta con workflow pruebas tanto para frontend y backend
    - Si pasan las pruebas se hace merge a main
